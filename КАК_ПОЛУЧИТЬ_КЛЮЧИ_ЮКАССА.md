# Как получить ключи для ЮKassa API

## 🔑 Получение ключей из личного кабинета ЮKassa

### Шаг 1: Вход в личный кабинет
1. Перейдите на https://yookassa.ru/my
2. Войдите в свой аккаунт

### Шаг 2: Получение Shop ID
1. В личном кабинете перейдите в раздел **"Настройки"** или **"Магазины"**
2. Найдите ваш магазин
3. **Shop ID** (идентификатор магазина) обычно отображается рядом с названием магазина
   - Пример: `390540012`
   - Это число без префиксов

### Шаг 3: Получение Secret Key
1. В личном кабинете перейдите в раздел **"Настройки" → "Секретный ключ"** или **"API"**
2. Найдите раздел с секретными ключами
3. Скопируйте **полный секретный ключ**
   - Для продакшена: начинается с `live_` (например: `live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`)
   - Для тестирования: начинается с `test_` (например: `test_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`)
   - ⚠️ **ВАЖНО**: Копируйте ключ полностью, включая префикс `live_` или `test_`

### Шаг 4: Обновление run_bot.command

Откройте файл `run_bot.command` и замените значения:

```bash
export YOOKASSA_SHOP_ID="390540012"  # Ваш Shop ID
export YOOKASSA_SECRET_KEY="live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx"  # Ваш полный секретный ключ
```

**Пример правильной настройки:**
```bash
export YOOKASSA_SHOP_ID="390540012"
export YOOKASSA_SECRET_KEY="live_AbCdEf123456GhIjKl789012MnOpQr345678StUvWx901234"
```

## ⚠️ Важные моменты

1. **Не публикуйте секретный ключ** в открытых репозиториях
2. **Ключ должен быть полным** - включая префикс `live_` или `test_`
3. **Shop ID** - это просто число, без префиксов
4. Для **тестирования** используйте ключ с префиксом `test_`
5. Для **продакшена** используйте ключ с префиксом `live_`

## 🔍 Проверка правильности ключей

После настройки ключей:
1. Запустите бота: `./run_bot.command`
2. Попробуйте создать бронирование
3. Если видите ошибку 401 - проверьте:
   - Правильность Shop ID
   - Что Secret Key скопирован полностью (включая префикс)
   - Что нет лишних пробелов в ключах

## 📞 Если не можете найти ключи

1. Обратитесь в поддержку ЮKassa
2. Или создайте новый секретный ключ в разделе "Настройки → API"
