# 🎵 MACHATA Studio Bot

Полнофункциональный Telegram бот для бронирования студии звукозаписи с поддержкой VIP пользователей, системой скидок и валидацией.

## ✨ Возможности

- 🎙 **Бронирование студии** (самостоятельная работа или со звукорежем)
- 🎸 **Репетиционная комната** (час за часом)
- 👑 **VIP система** с персональными скидками
- 💚 **Умные скидки** (3+ часа: -10%, 5+ часов: -15%)
- ☎️ **Валидация номера телефона** (11 цифр)
- 📅 **Календарь бронирований** с пагинацией
- 💬 **Комментарии** к сеансам (опционально)
- ✅ **Отмена броней** с освобождением времени
- 📊 **JSON хранилище** для бронирований и конфига

## 🚀 Быстрый старт

### Требования
- Python 3.7+
- pip

### Установка

1. **Установи зависимости:**
```bash
pip install pytelegrambotapi
```

2. **Скачай бота** и отредактируй:
```python
API_TOKEN = 'ТУТ_ВСТАВЬ_ТОКЕН_ОТ_BOTFATHER'
```

3. **Запусти бота:**
```bash
python machata_bot.py
```

## 📝 Конфигурация

### Тарифы
Отредактируй `DEFAULT_CONFIG`:
```python
'prices': {
    'repet': 700,      # Репетиция
    'studio': 800,     # Аренда студии (самостоятельно)
    'full': 1500,      # Со звукорежем
}
```

### VIP пользователи
Добавь в `VIP_USERS`:
```python
VIP_USERS = {
    123456789: {'name': 'Иван', 'discount': 20},
}
```

Как узнать ID? Запусти бота, напиши /start, посмотри в консоль.

### Режим работы
```python
'work_hours': {'start': 9, 'end': 22},  # 9:00 - 22:00
'off_days': [5, 6],                     # Сб (5), Вс (6)
```

## 📞 Контакты в боте

Отредактируй переменные в начале файла:
```python
STUDIO_CONTACT = "+7 (977) 777-78-27"
STUDIO_ADDRESS = "Москва, Загородное шоссе, 1 корпус 2"
STUDIO_TELEGRAM = "@majesticbudan"
```

## 🎮 Команды

- `/start` — Начать бронирование
- `🎙 Запись трека` — Аренда студии
- `🎸 Репетиция` — Репетиционная комната
- `📝 Мои бронирования` — Список твоих броней
- `💰 Тарифы & акции` — Прайс лист
- `📍 Как найти` — Адрес и карты
- `💬 Живой чат` — Связь с студией

## 📊 Файлы данных

- `machata_bookings.json` — Все бронирования (создаётся автоматически)
- `machata_config.json` — Конфигурация (опционально, иначе используется DEFAULT_CONFIG)

## 🔧 Особенности

✅ **Валидация номера** — только 11 цифр  
✅ **Система скидок** — VIP или по количеству часов  
✅ **Умное расписание** — выходные исключаются автоматически  
✅ **Отмена броней** — с освобождением времени  
✅ **Пропуск комментария** — кнопка "⏭️ Пропустить"  
✅ **4-шаговый процесс** — логичный и интуитивный  

## ⚠️ Важно

- Бот хранит данные в JSON локально (для продакшена используй БД)
- Все времена в часах (00:00 - 23:00)
- Названия сервисов можно менять в диктионарях `names`
- Текст сообщений полностью кастомизируется
